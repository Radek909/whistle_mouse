Mouse

Whistle Mouse

LIST FUNCTION



TYPE:


CODE ; NAME; FUNCTION


BACKWARD – moves the cursor or Rewind


FORWARD – moves the cursor or scroll forward


STANDBY – Enables mouse simulations and turns off sound / and on retreat


FLIP – converts cursor movement from horizontal to vertical / i to retreat


SCROLL – change cursor movement to scroll movement


MOUSE_CLICK – left mouse button


DOUBLE_MOUSE_CLICK – double click left mouse button


RIGHT_MOUSE_CLICK – right mouse button


MOUSE_CLICK_HOLD – holding down the left mouse button


RIGHT_MOUSE_CLICK_HOLD – holding down the right button Mouse


MAX_BACKWARD – cursor or scroll all the way back


MAX_FORWARD – cursor or scroll as far forward as possible


STANDBY0 – no mouse simulation


STATEDBY1 – mouse simulation


MUTE – mutes or restores the volume level


MUTE0 – mutes


MUTE1 – amplifies


VOL- - mutes by 0.1


VOL+ - turns up the volume by 0.1


CORD – moves the cursor to the specified X coordinates and Y e.g.: CORD-100,100


FILE – opens the file from the given location, e.g.: FILE-NOTEPAD


WAIT – chaka given time in ms. e.g.: WAIT-1000 (waits 1 second)


SENDKEY – enters a virtual string of characters, e.g.; SENDKEY-TEST


{TAB} {ENTER} {BACKSPACE} {BREAK} {CAPSLOCK} {DELETE} {DOWN} {UP} {LEFT}

{RIGHT} {END} {ESC} {HELP} {HOME} {INSERT} {NUMLOCK} {PGDN} {PGUP} {PRTSC}

{SCROLLLOCK}{F1}{F2}....+ SHIFT^CTRL % ALT


T_BOX – opens a tool box window


T_BOX_CLOSE – closes the toolbox window



TYPE:


FUNCTION; PARAMETER; PARAMETER


T_BOX – adds functions to the tool box, e.g.: T_BOX; NAME; FUNCTION (functions above)


F_TIME – functions that set the time. There are 3 :


- FRQ3_LONG – number of cycles expressed in refresh rate (z SETTINGS) after how many function 3 will work,

dintentionally set as STAND-BY



-FRQ_SHORT and FRQ_TIMER – set the time of operation of the second type calling a function

(read: CALLING FUNCTIONS:)

SHORT – turns on the listening



TIMER – time Listen


Przykład:



F_TIME; FRQ3_LONG;35


F_TIME; FRQ_SHORT;20 // max cycles above no Work



F_TIME; FRQ_TIMER;45 max. listening cycles above turns off


MOUSE – millisecond / pixels. Determines how many pixels to move the cursor in a given time interval. Example:


MOUSE;160;10

MOUSE;120;8

MOUSE;80;5

MOUSE;60;4

MOUSE;40;3 from 40 to 60 c o 3 pixels etc

MOUSE;10;2 from 10 to 40 every 2 pixels

MOUSE;0;1 from 0 to 10 every 1 pixel


SCROLL - ms / jump. Determines how many jumps will be performed at a given time.

Example:


SCROLL;120;4 4 jumps every 120ms

SCROLL;50;6 6 strokes every 50 ms

SCROLL;20;7

SCROLL;1;1


TBOX_SPEED – same principle as in SCROLL
