Settings:

























Important features to discuss are:


- Buffer – this number determines the number of registered frequencies that must be in one range for it to work function. The default number is 8.


- Refresh rate in ms – the speed at which it occurs re-recording the frequency.




Functions: (list of functions in the FUNCTION LIST tab)






















Each function is 1 line.


The syntax is as follows:


CODE; NAME; FUNCTION

(parameters separated by a semicolon)


for example.:


1 ; name (any) or constant, e.g.: (LNG_1 – depending on selected language) ; BACKWARD


OR


FUNCTION; PARAMETER; PARAMETER


(list of functions in the "List of Functions" tab)


Functions can be combined with /*/ e.g.:


32; NAME; FLIP/*/FLIP – will make a double function call
